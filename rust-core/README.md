# pyferox-rust

Rust acceleration core for [PyFerOx](https://github.com/pablo0723/pyferox).

`pyferox-rust` provides compiled runtime primitives used by the Python framework:

- schema compilation and validation
- fast query/path parsing
- JSON serialization
- low-overhead Python bindings via `pyo3`

This package is intended to be consumed by `pyferox`, but can also be used directly in advanced setups.

Repository: https://github.com/pablo0723/pyferox
